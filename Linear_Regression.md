## 🧠 System Overview

> Initial step toward building machine intelligence.

This module simulates how a system learns a relationship between variables using:

* Trial-based parameter search
* Error minimization
* Model fitting

---

## ⚙️ Core Equation

```
y = wx + b
```

* **w** → weight (slope)
* **b** → bias (offset)

---

## 🔁 Learning Mechanism

The system performs:

1. Generate candidate parameters (w, b)
2. Compute predictions
3. Evaluate error
4. Compare across all combinations
5. Select optimal configuration

---

## 📉 Error Function

```
Error = (y_true - y_pred)^2
```

Total error is accumulated across dataset.

---

## 🔬 Optimization Strategy

> Exhaustive Search (Discrete Parameter Space)

* w ∈ [-10, 10]
* b ∈ [-10, 10]

System evaluates all combinations.
