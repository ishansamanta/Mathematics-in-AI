 Linear Regression From scratch using Brute Force Method and without any libraries
 The code in linear_regression.py file is used to find the best fitting linear equation of the form y=wx+b, within a very limited dataset and minimizing the prediction error by a brute search approach.
 Dataset:
 | x   | y_true|
 | -   |    -- |
 | 1   |    2  |
 | 2   |    4  |
 | 3   |    6  |
 | 4   |    8  |
 | 5   |    10 |
 | 6   |    12 |
 The dataset follows a linear relationship where y increases proportionally to x.
 We assume a linear model:
 (y = wx + b)
 Generating Predictions
 For each data point compute the predicted value using current w and b
 Calculate Error: (y_true-y_pred)^2
 [Then sum over all data points.]
 Brute Force Search:
 1.Try values of w from -10 to 10
 2.Try values of b from -10 to 10
 3.Compute total error for each pair
 4.Select the pair with minimum error

'''Brute-force search can find optimal parameters for small datasets
Error minimization is the core idea behind machine learning
This approach does not scale for large datasets or continuous values
Efficient methods like gradient descent are required for real-world problems'''
